# NeurofleetX Tech Stack

## AI Engine
- **Framework**: Spring Boot, Python (ML models)
- **Libraries**: TensorFlow, Scikit-learn, OpenAI API
- **Services**: Route optimization, traffic prediction, predictive maintenance
- **Data Processing**: Apache Kafka (event streaming), Apache Spark (batch processing)
- **Storage**: PostgreSQL, Redis (caching)

## Maps & Geolocation
- **Frontend**: Leaflet.js, Mapbox GL JS
- **Backend**: Google Maps API, Mapbox API
- **Real-time Tracking**: WebSocket, Socket.io
- **Database**: PostGIS (geospatial queries)

## Communication
- **API**: REST (Spring Boot REST Controllers)
- **Real-time**: WebSocket, Socket.io
- **Messaging**: Apache Kafka, RabbitMQ
- **Notifications**: Firebase Cloud Messaging (FCM), SendGrid (email)
- **Auth**: JWT (JSON Web Tokens), OAuth 2.0

## Deployment
- **Infrastructure**: Docker, Kubernetes (K8s), Docker Compose
- **Cloud Platforms**: AWS (EC2, RDS, S3), Azure (App Service, Container Apps), Google Cloud (Compute Engine, Cloud SQL)
- **CI/CD**: GitHub Actions, Jenkins, GitLab CI/CD
- **Monitoring**: Prometheus, Grafana, ELK Stack
- **Load Balancing**: Nginx, HAProxy, AWS ALB
- **Database**: MySQL, PostgreSQL, MongoDB

## Frontend Stack
- React 19, TypeScript, Vite
- Tailwind CSS, Shadcn UI
- React Router, React Hook Form
- Recharts (charts), Leaflet (maps)

## Backend Stack
- Java 17+, Spring Boot 3.x
- Maven (build), JPA/Hibernate (ORM)
- MySQL/PostgreSQL (database)
- Lombok, Validation (spring-boot-starter-validation)

## Security
- SSL/TLS, HTTPS
- OAuth 2.0, JWT tokens
- API rate limiting
- Data encryption (AES-256)

## Monitoring & Logging
- ELK Stack (Elasticsearch, Logstash, Kibana)
- Prometheus + Grafana
- Splunk, Datadog
- Application Insights (Azure)

## Version Control
- Git, GitHub
